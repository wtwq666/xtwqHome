📖 Git 操作指南

一份简洁实用的 Git 入门与常用操作手册

1. 配置 Git 用户信息

git config --global user.name "你的名字"
git config --global user.email "你的邮箱@example.com"

2. 克隆仓库

git clone https://github.com/用户名/仓库名.git

3. 日常工作流程

  • git status — 查看仓库状态
  • git add . — 添加所有改动到暂存区
  • git commit -m "说明" — 提交更改
  • git push — 推送到远程仓库

4. 分支操作

git branch 新分支名          # 创建分支
git checkout 新分支名        # 切换分支
git checkout -b 新分支名     # 创建并切换
git merge 新分支名           # 合并分支

5. 拉取远程更新

git pull

6. 小贴士

  • 提交信息要清晰，例如 fix: 修复登录按钮样式
  • 勤提交、勤推送，避免大量改动导致冲突
  • 使用 .gitignore 忽略不需要追踪的文件
